## 소개
SibalScript는 한국인먼의 뜨거운 정신이 살아숨쉬고있는 우리 민족만의 정겨운 언어입니다.
<br/>

## 문법

```python
# 표준 SibalScript 문법을 준수하는 스크립트는 아래와 같이 "시발놈아!"로 시작됩니다.
시발놈아!

...

수고해라!
# 표준 SibalScript 문법을 준수하는 스크립트는 아래와 같이 "수고해라!"로 끝마칩니다.
```

```
봐라씨발("안녕")
```
